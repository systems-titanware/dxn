## Features Grid

| Feature | Description |
|---------|-------------|
| **End-to-End Encryption** | Your data is encrypted before it leaves your device. Not even we can read it. |
| **Offline-First** | Full access for up to 30 days without internet. No connectivity anxiety. |
| **Single Device Security** | Your keys are bound to your device. No password alone can compromise your data. |
| **Recovery Without Backdoors** | Recovery codes you control. No "forgot password" that bypasses security. |
| **Self-Hosted Option** | Run Vault on your own hardware if you prefer. Full sovereignty. |
| **Open Source** | Audit the code yourself. Security through transparency. |

## Trust Section

**Section Title:**
Security You Can Verify

**Body Text:**
Vault uses the same encryption trusted by banks and governments:

- **Argon2id** password hashing (winner of Password Hashing Competition)
- **Ed25519** signatures (used by SSH, Signal)
- **AES-256-GCM** encryption (bank-grade)
- **Double Ratchet** protocol (Signal's encryption standard)

We don't ask you to trust us. We ask you to verify.

**Image Suggestion:**
Lock/shield icon with encryption algorithm names subtly visible, or a "security verified" badge style graphic.
